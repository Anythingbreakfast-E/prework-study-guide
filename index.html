HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The metadata contains information about accessibility, search engines, and performance
  • The child elements are nested inside other elements
  • The parent element holds all the child elements and can be the body
  • Semantic HTML elements describes its content and the role it plays on the webpage
  • Elements contain an opening tag, content, and closing tag
  • Tags are what is in between angle brackets

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • External Style Sheet is the most commonly type of CSS used
  • Use a class attribute to only add styles to certain elements you want

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • git commands create a repo, add code to repo, and view the different versions of our code
  • Github is a webinterface for Git and is version control software and allows multiple people to work at the same time on a program

Javascript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • Javascript is case sensitive
  • A string variable is a sequence of text and had quote marks
  • A number variable is a number that doesn't have quotes
  • A Boolean variable is True/False with no quotes
  • A array variable allows you to store multiple values in a single reference
  • Comments are snippets of text that can be added along with code
  • Operators is a mathematical symbol that produces a result based off 2 values
  • Conditionals are code structues used to test if an expression returns true or not
  • Functions are a way of packaging functionality that you wish to reuse
  • Arrays are used when we want to perform a task on multiple pieces of data at once
  • console.log method is used to output a message to the web console by adding an argument
  • [for] is used for looping
  • Starting point declares a variable and has a value of 0
  • Second statement is the condition
  • Final statement allows the array to iterate over each item
  • [c++] = x +1
  • Function is a set of instructions that tells the computer how to perform a certain task
